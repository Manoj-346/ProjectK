Web Development Terms

HTML
The standard markup language for documents designed to be displayed in a web browser. It is the foundation of all web pages.
CSS
Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language like HTML. It handles the colors, layouts, and fonts.
JavaScript
A programming language that is one of the core technologies of the World Wide Web. It enables interactive web pages and is an essential part of modern web applications.
Front-end
The part of the website that the user interacts with directly. It includes everything from the text and images to the buttons and forms. It is also known as the client-side.
Back-end
The server-side of a website. It is the part of the website that users don't see, responsible for storing and organizing data, and ensuring the front-end works correctly. It is also known as the server-side.
